# module-for-shapes

Тестовое задание - Vue3.js

### Разработчик: Шахворостов Сергей

#### Для запуска проекта

```
"dev": "vite"

"build": "vite build"

"serve": "vite preview"
```

```
yarn install
```

### Compiles and hot-reloads for development

```
yarn serve
```

### Compiles and minifies for production

```
yarn build
```

### Lints and fixes files

```
yarn lint
```
